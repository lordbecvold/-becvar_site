getVisitorStatus()

Retrieves the status of a visitor with the given ID.

public getVisitorStatus(int $id) : string

This method constructs a cache key for the specified visitor ID, retrieves the status from the cache manager, and returns the status if found. If the status is not found in the cache or if it's offline, 'offline' is returned.

Parameters
$id : int

The ID of the visitor.

Return values
string —

The status of the visitor ('online' if online, 'offline' if not found or offline).
